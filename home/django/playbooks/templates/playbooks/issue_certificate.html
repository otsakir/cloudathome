Issue certificate

Playbooks | Dashboard

Obtains a TLS certificate via Let's Encrypt for a domain you own. DNS must already point to the cloud server.

  1. Registers a temporary HTTP proxy mapping on the cloud server
  2. Opens an SSH tunnel to a local certbot port
  3. Runs certbot (HTTP-01 challenge through the tunnel)
  4. Removes the temporary proxy mapping and tunnel

Note: This may take up to 60 seconds. Do not navigate away while it is running.
